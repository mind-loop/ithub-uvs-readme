# 🚀 VibeCoding Portfolio — AI Prompt Guide

> **Workflow:** GitHub repo → README дээр "." дар → github.dev нээ → index.html үүсгэ → AI prompt өг → Deploy

---

## ⚡ Step 1 — GitHub Repository

1. [github.com](https://github.com) → **New repository**
2. Name: `my-portfolio` · ✅ Public · ✅ Add README
3. **Create repository**
4. Гарын дэвтэр дээр **`.`** дар → github.dev нээгдэнэ
5. New File → `index.html` үүсгэ

---

## 🤖 Step 2 — AI Prompt

> Доорх prompt-г **бүгдийг** copy хий → `[...]` хэсгийг бөглө → ChatGPT-д paste хий

## 💾 Step 3 — Файл хадгалах & Push

1. AI-н кодыг **бүгдийг** copy хий
2. github.dev → `index.html` → paste → **Ctrl+S**
3. Source Control `⎇` → **+** → Commit message: `Add portfolio` → **Commit & Push**

---

## 🚀 Step 4 — Vercel Deploy

1. [vercel.com](https://vercel.com) → **Sign up with GitHub**
2. **Add New Project** → `my-portfolio`
3. Framework: **Other**
4. **Deploy** → 30 секунд → URL авна 🎉

**Таны URL:** `https://my-portfolio.vercel.app`
