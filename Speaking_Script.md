## Slide 2 · "Та .docx явуулдаг уу?"
**⏱ 0:05 – 0:12 · 7 минут**

*[Оролцогчдыг хар. Асуулт тав.]*

> Нэг асуулт.
> Өнгөрсөн сард ажлын файл, CV, ямар нэг зүйлийг
> .docx болгон имэйлээр явуулсан хүн гараа өргөнө үү?

*[Хүлээ. Гар өргөгдөнө. Толгой дохи.]*

> Тийм. Бараг бид бүгд тийм.

> Өнөөдрийн дараа та нар тэр файлын оронд ЛИНК явуулна.
> Тэр линк таны нэрийн хуудас. Таны portfolio. Таны бодит вэбсайт.

*[Зүүн карт руу заа.]*

> Файл явуулдаг хүн ингэж харагддаг —
> project_final_v3_REAL.docx.
> Laptop эвдэрвэл бүх зүйл алга. Шинэчлэх боломжгүй.

*[Баруун карт руу заа.]*

> Линк явуулдаг хүн өөр мессеж илгээдэг.
> "Энэ хүн мэргэжлийн. Орчин үеийн. Итгэлтэй."
> Ажил хайхад HR яг үүнийг хайдаг.

## Slide 3 · GitHub = Ажлын түүх · VS Code = Ажлын өрөө
**⏱ 0:12 – 0:20 · 8 минут**

*[Зургийг харуул. 3 секунд хүлээ — хүмүүс өөрсдөө харах цаг өг.]*

> Зүүн талд юу харж байна?

*[Хариулт хүлээ.]*

> Дэвтэр. GitHub бол яг л тэр.
> Таны ажлын намтар.
> Өөрчлөлт бүр хуудас болж хадгалагдана.
> Буцааж авна. Харьцуулна. Хэн юу хийснийг мэдэнэ.

*[Баруун тал руу заа.]*

> Баруун талд таны ажлын өрөө — VS Code.
> Код бичих, алдаа шалгах, AI туслагч — бүгд нэг дор.
> Хөтчөөр ч нээгдэнэ, laptop дээр ч суулгана.

*[Товч зогс.]*

> Google Drive файл хадгалдаг.
> GitHub өөрчлөлт бүрийн ШАЛТГААНЫГ хадгалдаг.
> Энэ ялгааг ойлгосон хүн мэргэжлийн болсон гэсэн үг.

> Нэг GitHub аккаунтаар хаана ч ажиллана.
> Кафед байсан ч, гэртээ байсан ч, өөр компьютер дээр ч.
