## 📍 НЭЭЛТ [0:00 – 0:05]

**[Laptop нээж, GitHub хуудсыг дэлгэцэнд харуул. 5 секунд чимээгүй бай.]**

> "Та нарын хэд нь өнгөрсөн сард ажлын файл, CV, ямар нэг зүйлийг .docx эсвэл .pdf болгон явуулсан бэ?"

**[Гараа өргүүл гэж хэл. Хүлээ. Бараг бүгд өргөнө.]**

> "Өнөөдрийн дараа та нар тэр файлын оронд ЛИНК явуулна. Тэр линк таны нэрийн хуудас, portfolio, бодит вэбсайт байна."

**[Нэг секунд чимээгүй байна.]**

> "Бид өнөөдөр зөвхөн 'вэб хийх' биш — мэргэжлийн хөгжүүлэгчдийн яг ашигладаг workflow-г эзэмшинэ. GitHub дээр код хадгалах, AI-тай хамт хурдан хөгжүүлэх, Vercel-ээр нэг товшилтоор дэлхийд нийтлэх. Энэ бүхнийг 1 цагийн дотор та нарын нүдний өмнө хийж харуулна."

**[GitHub дээр "." дар — VS Code хөтчөөр нээгдэнэ.]**

> "Харав уу? Тэр '.' товч л дарахад мэргэжлийн хөгжүүлэлтийн орчин нээгдчихлээ. Суулгах зүйл алга. Өнөөдөр бид яг үүнийг хийнэ."

## 📍 ОЙЛГОЛТ [0:05 – 0:15]

**[Slide 3 харуул]**

> "GitHub бол Google Drive биш."

> "Google Drive файл хадгалдаг — GitHub харин өөрчлөлт бүрийн ШАЛТГААНЫГ хадгалдаг. Хэн өөрчилсөн, хэзээ, яагаад — бүгд үлдэнэ. Буцах товч үргэлж байна."

> "VS Code — код бичих + алдаа шалгах + AI туслагч — бүгд нэг дороо. Та хоёр горимоор ашиглаж болно: компьютер дээр суулгасан хувилбар, эсвэл хөтчөөр нээдэг хувилбар."

> "Ажил хайхад CV .docx явуулдаг хүн vs. GitHub линк явуулдаг хүн — хоёр дахь нь 'энэ хүн мэргэжлийн' гэсэн мессеж илгээнэ."

## 📍 SETUP [0:15 – 0:25]

> "Одоо бүгд репозиторидоо ор. Гарын дэвтэр дээр '.' дар."

**[Хүлээ — бүгд VS Code нээгдэхэд хариу үйлдэл гарна.]**

> "Ингэж та хөтчийн tab нээхэд л мэргэжлийн хөгжүүлэлтийн орчин бэлэн болчихно."

## 📍 ПРАКТИК [0:25 – 0:40]

**[Slide 5 харуул]**

> "Одоо ViteJS төсөл эхлүүлнэ. VS Code-ийн terminal нээнэ — Ctrl + backtick."

**[Terminal нээж командуудыг бич. Дэлгэцийг томруул.]**

```
npm create vite@latest my-portfolio -- --template vanilla
cd my-portfolio
npm install
npm run dev
```

> "localhost:5173 нээхэд хуудас харагдана. Энэ бол таны хөгжүүлэлтийн орчин — код өөрчлөх бүрт автоматаар шинэчлэгдэнэ."

**[AI prompt өг.]**

> "Одоо AI-д хэлнэ — 'Надад portfolio хуудас хийгээд өг.' Гарсан кодыг index.html-д тавь. Хаягаа, мэргэжлийнхаа нэрийг орлуул."

**[5 минут оролцогчид өөрөө хийх цаг өг.]**

## 📍 LIVE DEMO [0:40 – 0:55]

**[Slide 6 харуул]**

> "Утсаа гаргацгаана уу. Энэ линкийг нааш нь байлга."

**[QR эсвэл URL явуул. Хүлээ.]**

> "Одоо та бүхэнд бодит хөгжүүлэлтийн ид шидийг харуулна. Би дэлгэц дээр нэг өөрчлөлт хийнэ, GitHub-руу push хийнэ — 10 секундын дараа таны утсан дээрх сайт шинэчлэгдсэн байна."

**[index.html-д нэг мөр өөрч: өнгө эсвэл текст.]**

> "Харсан уу? Зөвхөн нэг мөр. Одоо GitHub-руу явуулна."

**[Terminal дээр:]**
```
git add .
git commit -m "Өнгө өөрчиллөө"
git push
```

> "Push хийгдлээ. Vercel одоо автоматаар deploy хийж байна... та нар утсаа харцгаана уу."

**[ЧИМЭЭГҮЙ БАЙ. 10-15 секунд хүлээ. Оролцогчид "шинэчлэгдлээ!" гэж хэлэх болно.]**
