## 📍 SLIDE 1 — Гарчиг [0:00 – 0:05]

**[Laptop нээ. GitHub хуудсаа дэлгэцэнд харуул. 5 секунд чимээгүй бай.]**

> *"Та нарын хэд нь өнгөрсөн сард ажлын файлыг .docx болгон явуулсан бэ?"*

**[Гараа өргүүл гэж хэл. Хүлээ.]**

> *"Өнөөдрийн дараа та нар тэр файлын оронд ЛИНК явуулна. Тэр линк таны нэрийн хуудас, portfolio, бодит вэбсайт байна."*

**[GitHub дээр "." дар — VS Code нээгдэнэ. Хэн нэгэн "вау" гэх болно.]**

> *"Харав уу? Тэр '.' товч л дарахад мэргэжлийн орчин нээгдчихлээ. Суулгах зүйл алга. Өнөөдөр яг үүнийг хийнэ."*

⏱ **5 минут**

---

## 📍 SLIDE 2 — .docx vs Линк [0:05 – 0:12]

> *"Файл явуулах нь яагаад хуучирсан арга болсон бэ?"*

**[Зүүн карт руу заа.]**

> *"project_final_v3_REAL.docx — энэ нэрийг бид бүгд мэднэ. Laptop эвдэрвэл бүх зүйл алга. Хэн юу өөрчилснийг мэдэхгүй."*

**[Баруун карт руу заа.]**

> *"GitHub линк явуулдаг хүн өөр мессеж илгээдэг: 'Энэ хүн мэргэжлийн, шинэчлэгдсэн, аюулгүй.' Ажил хайхад HR хүмүүс яг үүнийг хайдаг."*

⏱ **7 минут**

---

## 📍 SLIDE 3 — GitHub = Түүх · VS Code = Өрөө [0:12 – 0:20]

**[Зургийг харуул. Тайлбарлахаас өмнө хэдэн секунд хүмүүс өөрсдөө харах цаг өг.]**

> *"Зүүн талд юу харж байна? Дэвтэр. GitHub бол яг тэр — таны ажлын намтар. Өөрчлөлт бүр 'хуудас' болж хадгалагдана. Буцааж авна, харьцуулна, хэн юу хийснийг мэдэнэ."*

> *"Баруун талд — таны ажлын өрөө. VS Code. Дэлгэц, гар, AI туслагч — бүгд нэг дор. Хөтчөөр ч нээгдэнэ, laptop дээр ч суулгана. Гол нь: нэг GitHub аккаунтаар хаанаас ч ажиллана."*

> *"Google Drive файл хадгалдаг. GitHub өөрчлөлт бүрийн ШАЛТГААНЫГ хадгалдаг. Энэ ялгааг ойлгосон хүн мэргэжлийн болсон гэсэн үг."*

⏱ **8 минут**

---

## 📍 SLIDE 4 — Орчин бэлдэх [0:20 – 0:30]

**[Бүгдийг дагуулж хий — слайдын 4 алхмыг зэрэг гүйцэтгэнэ.]**

> *"Одоо та нар надтай хамт хийнэ. Laptop нээцгээе."*

**Алхам 1:**
> *"github.com руу ор. Sign up. Username нэрлэхдээ анхаар — энэ таны URL болно. github.com/таны-нэр. Товч, мэргэжлийн, тоогүй."*

**[2 минут хүлээ.]**

**Алхам 2:**
> *"New repository. my-portfolio. Public. README нэм. Create."*

**Алхам 3 — ТРЮК:**
> *"Одоо репозиторидоо ор. Гарын дэвтэр дээр '.' дар."*

**[Хүлээ. Хариу үйлдэл гарна.]**

> *"VS Code нээгдлээ. Terminal хэрэггүй. Framework хэрэггүй. Энэ бол таны ажлын орчин."*

**Алхам 4:**
> *"README.md нээцгээе. Доторх prompt-г харна уу — дараа нь ChatGPT-д өгнө."*

⏱ **10 минут**

---

## 📍 SLIDE 5 — AI Prompt [0:30 – 0:45]

> *"Prompt-г copy хийнэ. [...] хэсгийг өөрийн мэдээллээр бөглөнө."*

**[2-3 минут өгч оролцогчид бөглүүл.]**

> *"Нэрээ, мэргэжлийнхаа нэрийг, 2-3 өгүүлбэр танилцуулга бич. Skill-ээ нэм. Бэлэн болсон уу?"*

**[ChatGPT нээ.]**

> *"Prompt-г paste хий. Enter. AI index.html код бичнэ."*

**[Хүлээ — 30-60 секунд.]**

> *"Кодыг бүгдийг нь copy хий. github.dev дээр New File → index.html. Paste. Ctrl+S. Source Control → + → Commit message: 'Portfolio эхний хувилбар' → Commit & Push."*

> *"GitHub дээр index.html файл харагдаж байна уу? — Амжилтлаа."*

⏱ **15 минут**

---

## 📍 SLIDE 6 — Deploy [0:45 – 0:52]

> *"Одоо дэлхийд харуулах цаг. Vercel."*

> *"vercel.com → Sign in with GitHub → Add New Project → my-portfolio → Deploy."*

**[30-60 секунд хүлээ.]**

> *"URL авлаа. my-portfolio.vercel.app. Энэ URL-г утсандаа нээцгээе."*

**[Бүгд нээхийг хүлээ.]**

> *"Харж байна уу? Таны нэрийн хуудас. Дэлхийн хаана ч байгаа хүн энэ линкийг нээхэд харна."*

> *"Энэ URL-г CV-дээ, LinkedIn-д, имэйлийн signature-д тавь. .docx биш — линк."*

⏱ **7 минут**

---

## 📍 SLIDE 7 — Live Demo [0:52 – 1:00]

**[URL-г бүгдэд мессенжерт явуул. Утсаа гаргахыг хүс.]**

> *"Одоо та бүхэнд бодит ид шидийг харуулна. Би нэг өөрчлөлт хийнэ — 10 секундын дараа таны утсан дээр шинэчлэгдэнэ."*

**[index.html-д нэг мөр өөрч — өнгө эсвэл текст.]**

> *"Харсан уу? Нэг мөр. Одоо push хийнэ."*

**[Terminal биш — Source Control → Commit → Push.]**

**[ЧИМЭЭГҮЙ БАЙ. 10-15 секунд хүлээ.]**

**[Оролцогчид "шинэчлэгдлээ" гэж хэлэх болно.]**

> *"Энэ бол мэргэжлийн хөгжүүлэгчдийн өдөр бүр хийдэг зүйл. Та нар өнөөдөр үүнийг хийлээ."*

> *"GitHub = ажлын түүх. VS Code = ажлын өрөө. Vercel = дэлхийн дэлгэц. Гурав хамтдаа — таны дижитал эрх чөлөө."*

⏱ **8 минут**
